# Configurar Variables de Entorno en Vercel

Ve a tu proyecto en Vercel y configura estas variables de entorno:

## Variables Requeridas:

### 1. VITE_GOOGLE_CLIENT_ID
- Ve a: https://console.cloud.google.com/apis/credentials
- Copia tu Client ID de OAuth 2.0
- Pégalo en Vercel

### 2. VITE_API_BASE_URL
- URL de tu backend API
- Ejemplo: `https://tu-api-backend.com/api`

## Pasos en Vercel:

1. Ve a tu proyecto en Vercel Dashboard
2. Settings > Environment Variables
3. Agrega cada variable:
   - Name: `VITE_GOOGLE_CLIENT_ID`
   - Value: `tu-client-id-aquí`
   - Environments: Production, Preview, Development
4. Repite para `VITE_API_BASE_URL`
5. Redeploy tu proyecto

**IMPORTANTE:** Después de agregar las variables, debes hacer un nuevo deployment para que surtan efecto.
